## 堆

维护一个长度为 $k$ 的小顶堆  
堆在一些特殊的排序问题中可以把规模从 `nlogn` 降到 `nlogk`  

```python
import heapq
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        heap = []
        for num in nums:
            if len(heap) < k: heapq.heappush(heap, num)
            elif num > heap[0]:
                heapq.heappop(heap)
                heapq.heappush(heap, num)
        return heap[0]
```

## 快速选择

```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        def myFindKthLargest(nums, start, l, k):
            # 随机快速选择的基准
            pivot = random.randint(start, l-1)
            nums[pivot], nums[start] = nums[start], nums[pivot]
            
            # 快速选择
            idx_num = nums[start]
            left, mid, right = start+1, start+1, l-1
            while mid <= right:
                if nums[mid] > idx_num:
                    nums[mid], nums[right] = nums[right], nums[mid]
                    right -= 1
                elif mid >= left and nums[mid] < idx_num:
                    nums[mid], nums[left] = nums[left], nums[mid]
                    left += 1
                else: mid += 1

            # 二分
            # left 左边都 < idx_num, right 右边都 > idx_num, [left, right] 是idx_num
            if l-right <= k <= l-left+1: return idx_num
            elif l-right > k:  # 去右边找
                return myFindKthLargest(nums, right+1, l, k)
            else:
                return myFindKthLargest(nums, start+1, left, k-(l-left+1))
        
        
        return myFindKthLargest(nums, 0, len(nums), k)
```
